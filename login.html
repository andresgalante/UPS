Welcome to Aerogear Unified Push Server! The AeroGear UnifiedPush Server is a server that allows sending native push messages to different mobile operating systems. Currently the server supports Apple’s APNs, Google Cloud Messaging and Mozilla’s Simple Push. Version is xxx.

First step: Keycloack configurations

UnifiedPush Server uses Keycloak as an authentication and user management endpoint. You can either use a default instance that comes with UnifiedPush Server installation or you choose to select an existing or a standalone Keycloak installation (so you can use it as an authentication backend for more applications).
